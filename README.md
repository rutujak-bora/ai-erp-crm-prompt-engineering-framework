# AI-Powered ERP & CRM System Generation Prompt Framework

## Project Overview

This project demonstrates advanced prompt engineering techniques used to generate enterprise-grade ERP and CRM systems using Generative AI tools such as ChatGPT, Claude, Gemini, and Emergent AI.

The objective of this prompt framework is to help businesses rapidly generate software architecture, database schemas, user interfaces, workflows, dashboards, reports, APIs, and automation processes through structured AI instructions.

---

## Business Problem

Many organizations struggle to develop ERP and CRM solutions because of:

* Complex requirements gathering
* Long development cycles
* High development costs
* Lack of technical documentation
* Poor workflow automation

This prompt framework addresses these challenges by enabling AI-assisted software generation.

---

## Master Prompt

Act as a Senior Software Architect, Business Analyst, UI/UX Designer, Product Manager, Database Architect, and Full Stack Developer.

Create a complete enterprise-grade ERP and CRM Management System for a growing organization.

The system must include:

### User Management

* Super Admin
* Admin
* Manager
* Employee
* Customer

### CRM Modules

* Lead Management
* Opportunity Tracking
* Customer Management
* Sales Pipeline
* Follow-Up Management
* Communication History

### ERP Modules

* Inventory Management
* Purchase Orders
* Sales Orders
* Vendor Management
* Customer Management
* Warehouse Management
* Finance & Accounting

### Dashboard Requirements

* Executive Dashboard
* Sales Dashboard
* Inventory Dashboard
* Customer Dashboard
* Financial Dashboard

### Database Requirements

Generate:

* ER Diagram
* Database Schema
* Relationships
* Primary Keys
* Foreign Keys

### Frontend Requirements

Create:

* Responsive Design
* Modern UI
* Mobile Compatibility
* Dark Mode Support

### Backend Requirements

Generate:

* REST APIs
* Authentication
* Authorization
* Audit Logs

### AI Features

* Predictive Analytics
* Demand Forecasting
* Smart Recommendations
* Automated Reporting

### Deliverables

Generate:

1. Complete SRS Document
2. System Architecture
3. Database Design
4. UI Wireframes
5. API Documentation
6. Development Roadmap
7. Testing Plan
8. Deployment Strategy
